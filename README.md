## Descrição
 O projeto consiste na composição de uma tabela que conta com capitais brasileiras e a distâncias entre essas capatais. As contantes 
 nesse projeto são:

 * ARACAJÚ
 * BELÉM 
 * BELO HORIZONTE
 * BOA VISTA
 * BRASILIA
 * CAMPO GRANDE 
 * CUÍABA
